🛠️ Email Sequence Builder
=========================

Define your audience and goal — get a full email sequence with subject lines, body copy, CTAs, timing, and an HTML preview.

## Install

```
https://raw.githubusercontent.com/qike7777/skill-library/main/skills/5-workflow-and-operations/email-sequence/SKILL.md
```

## Best For

- Marketers
- Founders
- Sales teams
- Content creators

## Input

- Target audience
- Campaign goal (onboarding, nurture, sales, re-engagement)
- Product or offer details

## Output

- Structured output based on your input

## Example Use Cases

- Build a 5-email welcome sequence for new signups
- Create a product launch drip campaign
- Write a re-engagement sequence for churned users

## How It Works

1. Sequence: define goal & audience → select sequence type → write all emails
2. Optimize: audit existing sequence → rewrite weak emails → A/B subject variants
3. Deliver: .md email copy + timing map + metrics tracking plan

## How to Use

1. Copy the Skill URL above and add it to your AI workspace
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result

## Tips

- Be specific about your goal for better output
- Include examples or context when possible
- Specify format preferences (table, bullets, prose, etc.)
- For best results, provide relevant source material upfront
