📈 Financial Analysis
=====================

![Category](https://img.shields.io/badge/category-Research%20&%20Analysis-lightgrey)

Build financial models, P&L, cash flow, unit economics, and investor materials. Auto-searches industry benchmarks. Delivers HTML dashboard + 4-sheet Excel workbook (Income Statement, Cash Flow, Unit Economics, Scenarios).

## Install

Add this skill to your CodeBanana workspace:

```

```
