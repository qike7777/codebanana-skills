🛠️ Excel Assistant
==================

Chat to create, edit, or batch-process Excel files — formulas, formatting, pivot tables, and data transformations.

## Install

```
https://raw.githubusercontent.com/qike7777/skill-library/main/skills/2-documents/excel-assistant/SKILL.md
```

## Best For

- Analysts
- Finance teams
- Operators
- Anyone who works with spreadsheets

## Input

- Describe what you need
- Existing Excel files
- Raw data or tables

## Output

- Structured output based on your input

## Example Use Cases

- Build a budget tracker with formulas from scratch
- Clean and restructure a messy data export
- Batch-process a folder of CSV files into a summary workbook

## How It Works

1. Build: describe table → agent generates config → run builder.py → deliver .xlsx
2. Edit: upload file → summarize → confirm changes → run editor.py → deliver .xlsx
3. Batch AI: upload file → pick column → define transform → run processor.py → deliver .xlsx

## How to Use

1. Copy the Skill URL above and add it to your AI workspace
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result

## Tips

- Be specific about your goal for better output
- Include examples or context when possible
- Specify format preferences (table, bullets, prose, etc.)
- For best results, provide relevant source material upfront
