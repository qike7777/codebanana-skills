🛠️ Build a Mobile App
=====================

![Category](https://img.shields.io/badge/category-Websites%20&%20Apps-lightgrey)

## How It Works
1. Design: mobile style research → design system (touch targets + safe areas + platform tokens) → user approval
2. Build: screen plan + navigation → TDD (Red→Green→Refactor) → code review → platform polish
3. Ship: Expo Go preview → QR code → TestFlight/Play Store build guide
