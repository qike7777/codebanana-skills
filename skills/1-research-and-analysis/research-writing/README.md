🔬 Deep Research Report
=======================

![Category](https://img.shields.io/badge/category-Research%20&%20Analysis-lightgrey)

Generate publication-quality research documents — papers, reports, literature reviews. Conducts web research, builds structured outline, writes full prose with citations, delivers HTML visual report + xlsx reference export.

## Install

Add this skill to your CodeBanana workspace:

```
Skill URL: https://raw.githubusercontent.com/qike7777/codebanana-skills/main/skills/1-research-and-analysis/research-writing/SKILL.md
```

## Best For

- Researchers
- Analysts
- Strategists
- Consultants

## Output

- Markdown report
- HTML visual report
- xlsx citation export

## How to Use

1. Install the skill using the Skill URL above
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result
