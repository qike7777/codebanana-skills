🛠️ Meeting Notes to Action Items
================================

Paste a meeting transcript — get a TL;DR, key decisions, action items, and a Word export in seconds.

## Install

```
https://raw.githubusercontent.com/qike7777/skill-library/main/skills/2-documents/meeting-summary/SKILL.md
```

## Best For

- Executives
- Project managers
- Teams of all sizes
- Remote workers

## Input

- Meeting transcript or rough notes
- Optional: participants, date, context

## Output

- Structured output based on your input

## Example Use Cases

- Turn a 1-hour Zoom transcript into a 1-page summary
- Extract action items and owners from a planning session
- Generate a shareable meeting recap for the whole team

## How It Works

1. Paste transcript/notes → generate minutes (Markdown) → auto-export .docx → optional follow-up research

4. Review and refine the result
