🛠️ Course Designer
==================

Input a topic and audience — get a full course with module outlines, lesson plans, exercises, quizzes, and an HTML course page.

## Install

```
https://raw.githubusercontent.com/qike7777/skill-library/main/skills/5-workflow-and-operations/course-designer/SKILL.md
```

## Best For

- Educators
- L&D teams
- Content creators
- Coaches
- Trainers

## Input

- Course topic
- Target audience and skill level
- Desired duration or depth

## Output

- Structured output based on your input

## Example Use Cases

- Design a 6-module online course on product management
- Build a workshop curriculum with exercises and assessments
- Create an onboarding training program for new hires

## How It Works

1. Design: define audience & goals → outline modules → write lesson plans
2. Content: develop slides + exercises + quizzes → assemble learner guide
3. Deliver: course page HTML + .md outline → ready for LMS upload

## How to Use

1. Copy the Skill URL above and add it to your AI workspace
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result

## Tips

- Be specific about your goal for better output
- Include examples or context when possible
- Specify format preferences (table, bullets, prose, etc.)
- For best results, provide relevant source material upfront
