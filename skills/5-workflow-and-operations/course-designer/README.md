🛠️ Design a Course
==================

![Category](https://img.shields.io/badge/category-Workflow%20&%20Operations-lightgrey)

Designs complete online courses, workshops, and training programs. Produces course outlines, lesson plans, learning objectives, exercises, quizzes, and learner guides.

## Install

Add this skill to your CodeBanana workspace:

```
Skill URL: https://raw.githubusercontent.com/qike7777/codebanana-skills/main/skills/5-workflow-and-operations/course-designer/SKILL.md
```

## How It Works
1. Design: define audience & goals → outline modules → write lesson plans
2. Content: develop slides + exercises + quizzes → assemble learner guide
3. Deliver: course page HTML + .md outline → ready for LMS upload
