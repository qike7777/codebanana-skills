# Social Media Strategist — SKILL.md

> Turn one idea into strong, platform-native content instead of posting the same thing everywhere.

---

## Overview

Every platform has its own grammar. The same idea expressed on LinkedIn feels wrong on TikTok. The job is to understand each platform's conventions and write content that belongs there.

**Core principle**: Platform-native > platform-agnostic. Adapt every piece for its channel.

---

## Phase 1 — Understand the Request

Before writing, collect:
- **Topic / idea**: What's the core message?
- **Goal**: Awareness, engagement, leads, community building?
- **Platforms**: Which channels? All of them or specific ones?
- **Brand voice**: Professional / casual / educational / entertaining / provocative?
- **Audience**: Who follows this account? What do they care about?
- **Existing content**: Any previous posts to match in style?

---

## Phase 2 — Platform Formats & Rules

### X / Twitter

**Character limits:**
- Tweet: 280 characters
- Thread: unlimited tweets

**What works:**
- Short, punchy takes with a clear point of view
- Threads for step-by-step breakdowns or long-form thinking
- Questions that invite replies
- Timely reactions to news/trends

**Thread structure:**
```
Tweet 1 (hook): Bold claim or surprising statement
Tweet 2-N (body): Each tweet = one idea, flows to next
Last tweet (CTA): Summary + ask (retweet, reply, follow)
```

**Hook formulas for X:**
- "I [did X unusual thing]. Here's what I learned:"
- "Unpopular opinion: [statement]"
- "Most people [do X wrong]. The fix is simple:"
- "[Number] things I wish I knew about [topic]:"
- "Stop [doing X]. Do this instead:"

---

### LinkedIn

**Post types:**
- Text post (max 3,000 chars — but 1,200 visible before "see more")
- Document carousel (PDF slides)
- Video
- Newsletter article

**What works:**
- Personal stories with professional lessons
- Frameworks and step-by-step systems
- Behind-the-scenes of work/process
- Data and insights from direct experience
- Opinions on industry trends

**LinkedIn post structure:**
```
Line 1 (hook): Single compelling sentence — make them click "see more"
[blank line]
Line 2-4 (context): Setup — why this matters
[blank line]
Main body: 3-7 bullet points OR short paragraphs
[blank line]
Closing: Lesson, takeaway, or question
[blank line]
CTA: Like / share / comment ask (optional but effective)
```

**LinkedIn hook formulas:**
- "I was wrong about [X] for years. Here's what changed my mind:"
- "After [doing X for Y years], here's the one thing that matters:"
- "[Number]-step framework for [outcome]:"
- "Hot take: [statement]"
- "Nobody talks about [X]. Let me explain why it matters:"

---

### Instagram

**Formats:**
- Single image post: image + caption
- Carousel: 2-10 slides (max engagement format)
- Reels: 15-90 seconds (< 30s for max reach)
- Story: 15s max, 24h lifespan

**Caption structure:**
```
Line 1 (hook): Must stand alone before "more" cutoff
Main copy: Story, tips, or explanation
Hashtags: 3-15 (end of post or first comment)
CTA: Save / share / comment / DM
```

**Carousel copy structure (10 slides max):**
```
Slide 1: Bold hook / title
Slides 2-8: One idea per slide, simple headline + 1-2 lines
Slide 9: Key takeaway
Slide 10: CTA + follow prompt
```

---

### TikTok / Reels / Shorts (Short Video Script)

**Structure (15-60s):**
```
0:00-0:03 (Hook): "Stop scrolling. [Bold claim or question]"
0:03-0:08 (Context): Why this matters / the problem
0:08-0:45 (Value): The content — tip, story, demo, breakdown
0:45-0:60 (CTA): "Follow for more. Comment [X] if you want [Y]"
```

**Hook types for short video:**
- Question: "Did you know [surprising fact]?"
- Controversy: "This is why [common belief] is wrong"
- Result: "I [achieved X] in [timeframe] — here's how"
- Challenge: "Try this for 7 days"
- POV: "POV: You just learned [X]"

**Script format:**
```markdown
[0:00] HOOK: [Exact words to say]
[0:03] CONTEXT: [Setup]
[0:08] POINT 1: [Say + visual]
[0:15] POINT 2: [Say + visual]
[0:22] POINT 3: [Say + visual]
[0:45] CTA: [Exact words]
```

---

### YouTube (Long-Form Script)

See `content-script-writer` skill for full YouTube scripts.

**YouTube-specific elements:**
- Title: SEO-optimized, 60 chars max, includes primary keyword
- Description: First 150 chars matter for SEO + CTR
- Chapters: Timestamp every 2-3 minutes for retention
- End screen: 20 seconds of CTAs (subscribe, related video)

---

### Newsletter / Email

**Structure:**
```
Subject: [Specific, personalized, creates curiosity — 40-50 chars]
Preview text: [Extends or complements subject — 90 chars]

Opening: 1-2 sentences that hook immediately
Main section(s): 1-3 topics, each with value
CTA: One clear next step
Sign-off: Personal, brief
```

**Subject line formulas:**
- "[Number] [things] to [outcome]"
- "The [adjective] way to [goal]"
- "Why [common belief] is costing you [cost]"
- "[Question]?"
- "Quick thought on [topic]"

## Phase 3 — Multi-Platform Repurposing

When given ONE source asset, adapt for all platforms:

```markdown
## Repurposing Plan: [Source Asset Title]

**Source**: [Blog post / video / podcast / talk / thread]
**Core idea**: [One sentence summary]

### Adaptation Map

| Platform | Format | Key Adaptation |
|---------|--------|---------------|
| X | Thread | [Breakdown into X tweets] |
| LinkedIn | Text post | [Professional framing + personal angle] |
| Instagram | Carousel | [Visual slides] |
| TikTok | Script | [Hook + punchy delivery] |
| Newsletter | Section | [Deeper context + personal story] |
```

**Adaptation rules:**
- Never copy-paste across platforms
- LinkedIn: more professional, first-person stories
- X: more opinionated, shorter takes
- Instagram: more visual descriptions, emotional hooks
- TikTok: more energetic, direct address ("you")
- Newsletter: more personal, deeper context

### Content Pillar Framework

Define 3-5 recurring content themes (pillars):

| Pillar | % of Content | Purpose | Example Topics |
|--------|-------------|---------|---------------|
| [Education] | 40% | Build authority | Tips, frameworks, how-tos |
| [Behind the scenes] | 25% | Build trust | Process, failures, decisions |
| [Opinion] | 20% | Drive engagement | Takes, debates, predictions |
| [Promotion] | 15% | Drive conversions | Products, services, offers |

## Phase 5 — Hashtag Strategy

### Hashtag Tiers

```
Tier 1 (Niche, 10K-100K posts): High relevance, less competition
  → Best for reaching your target audience
  → Example: #productmanagement (not #management)

Tier 2 (Medium, 100K-1M posts): Broader reach, moderate competition
  → Mix these in for visibility
  → Example: #startuplife

Tier 3 (Large, 1M+ posts): Maximum exposure, very competitive
  → 1-2 max per post
  → Example: #entrepreneurship
```

**Recommended mix**: 8-15 hashtags, 60% Tier 1, 30% Tier 2, 10% Tier 3

---

## Quality Gate

Before delivering any content:

- [ ] Does each post feel written FOR that platform, not adapted TO it?
- [ ] Does the hook stand alone — would someone stop scrolling?
- [ ] Is there one clear message per post (not 3)?
- [ ] Does the CTA tell the audience exactly what to do?
- [ ] No cross-platform copy-paste?
- [ ] Brand voice is consistent?
- [ ] No fabricated statistics or false claims?

## Theme Selection Guide

When generating the HTML output, **pick the theme that best matches the user's topic and audience**. Do not default to the same theme every time.

### Theme Options

| Theme | File Suffix | Visual Style | Best For |
|-------|-------------|--------------|----------|
| **Default** | `-default.html` | Vibrant modern — bright color, clean grids, multi-platform layout | Standard social media work, multi-platform content |
| **Dark** | `-dark.html` | Cyberpunk terminal — deep black, neon cyan/green, monospace fonts, scan-line overlay | Developer tools, technical docs, automation platforms, CLI-focused content, anything "engineering/hacker" |
| **Bold** | `-bold.html` | Neo-brutalism/Memphis — cream background, thick black borders, offset shadows, Space Mono, yellow+pink accents | Creative work, marketing, social/viral content, startups, anything that needs to feel energetic and memorable |

### Decision Rules

```
User's topic / audience                  → Recommended theme
────────────────────────────────────────────────────────────
Dev tools, APIs, coding, automation      → dark
Corporate SaaS, enterprise, formal       → default
Marketing, social, brand, creative       → bold
Data, analytics, dashboards, technical   → dark  
E-learning, education, professional docs → default
Startups, indie, viral, fun content      → bold
Writing, journalism, editorial content   → default
Music, gaming, culture, subculture       → dark or bold
Personal brand, portfolio, resume        → bold (if expressive) or default (if conservative)
```

### How to Apply

1. **Detect theme from user's context** — their topic, brand tone, or explicit preference
2. **Load the matching template file** (`script-collection-template-[theme].html`)
3. **Replace all `{VARIABLE}` placeholders** with generated content
4. **Deliver the file** — no build step required

If the user asks for a specific style or you're unsure, default to **default** and offer alternatives:
> "I used the clean default theme. Want to try the dark cyberpunk or bold neo-brutalism version?"

**① Markdown 文件** — 保存为 `script-[平台]-[序号].md`，格式：
```markdown

时长：XX-XX秒
风格：[风格描述]

[0:00] HOOK：
"..."
（画面：...）

[0:XX] CONTEXT：
"..."

[0:XX] POINT 1：
"..."
（画面：...）

[0:XX] CTA：
"..."
```

**② HTML 单篇展示页** — 使用 `script-single-template-default.html` (or `-dark` / `-bold`)，替换所有 `{{VARIABLE}}` 占位符，保存为 `script-[平台]-[序号].html`

**设计：** Neo-Brutalism — 奶油 `#f5f5f0` 底色，黄色 `#ffe600` accent，黑色 `3px` 粗边框，Space Mono 字体。左侧黑底时间轴导航，右侧报纸式台词排版，台词分三种样式（emphasis 大字 / question 黄色左边框 / soft 灰色），底部黑底 Markdown 原文区含复制按钮。
