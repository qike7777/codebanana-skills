🛠️ Meeting Summary
==================

![Category](https://img.shields.io/badge/category-Documents-lightgrey)

Paste a meeting transcript or rough notes — get structured minutes with TL;DR, decisions table, action items, and open questions. Auto-exports to Word. No API key needed.

## Install

Add this skill to your CodeBanana workspace:

```
Skill URL: https://raw.githubusercontent.com/qike7777/codebanana-skills/main/skills/2-documents/meeting-summary/SKILL.md
```

## Best For

- Writers
- Operators
- Legal teams
- Executives

## Output

- Structured output based on your input

## Capabilities
- Structured meeting minutes
- TL;DR summary
- Action items with owner/deadline/priority
- Decisions vs. open questions separation
- Word .docx export
- Quick format variants (30-second version, team email format)
- Optional follow-up research on topics mentioned

## How It Works
1. Paste transcript/notes → generate minutes (Markdown) → auto-export .docx → optional follow-up research

## How to Use

1. Install the skill using the Skill URL above
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result

## Tips

- Give clear context for better results
- Specify the format you want (e.g. "give me a table", "write as bullet points")
- Include examples when possible
- Use focused instructions for more reliable output
