🛠️ Contract Drafter
===================

Describe your terms — get a professionally structured contract, NDA, or employment agreement ready to review.

## Install

```
https://raw.githubusercontent.com/qike7777/skill-library/main/skills/2-documents/contract-drafter/SKILL.md
```

## Best For

- Founders
- Legal teams
- HR teams
- Freelancers
- Operators

## Input

- Contract type and key terms
- Parties involved
- Specific clauses or requirements

## Output

- Structured output based on your input

## Example Use Cases

- Draft an NDA for a new vendor partnership
- Generate a freelance services agreement from a project brief
- Create an employment contract template for a new hire

## How It Works

1. Draft: gather parties & scope → generate contract → explain key clauses
2. Review: analyze existing doc → flag risks → suggest revisions
3. Customize: select template → tailor terms → deliver .md + .docx

## How to Use

1. Copy the Skill URL above and add it to your AI workspace
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result

## Tips

- Be specific about your goal for better output
- Include examples or context when possible
- Specify format preferences (table, bullets, prose, etc.)
- For best results, provide relevant source material upfront
