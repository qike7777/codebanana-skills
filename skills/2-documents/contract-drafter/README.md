🛠️ Draft a Contract
===================

![Category](https://img.shields.io/badge/category-Documents-lightgrey)

Drafts, reviews, and explains contracts, NDAs, service agreements, terms of service, employment contracts, and other legal documents with jurisdiction-aware guidance.

## Install

Add this skill to your CodeBanana workspace:

```
Skill URL: https://raw.githubusercontent.com/qike7777/codebanana-skills/main/skills/2-documents/contract-drafter/SKILL.md
```

## Capabilities
- Contract
- NDA

## How It Works
1. Draft: gather parties & scope → generate contract → explain key clauses
2. Review: analyze existing doc → flag risks → suggest revisions
3. Customize: select template → tailor terms → deliver .md + .docx

## How to Use

1. Install the skill using the Skill URL above
2. Describe your task or paste your source material
3. Specify the output format you want
4. Review and refine the result
